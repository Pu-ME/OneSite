# 关于项目

该博客内容管理系统是基于SSM实现的一个个人博客系统，适合初学SSM和个人博客制作的同学学习。主要用到的技术包括Spring、SpringMVC、Mybatis、Jsp、mysql等。前段采用[Layui](https://www.layui.com)框架和[pinghsu](https://github.com/chakhsu/pinghsu)前台样式。
